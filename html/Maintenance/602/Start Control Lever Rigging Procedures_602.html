Refer to Pratt & Whitney Maintenance Manual for appropriate engine for general instructions and drawings. Rig Engine Start Control as follows:

  1. Position start lever in Cockpit in RUN position
  2. Adjust REB3N bearing at cable end so that condition lever on engine is straight up midway between stops.
  3. Pull start lever in cockpit all the way back against the quadrant stop. Condition lever on engine should be at the fuel cut-off position.
  4. Perform flight line engine runup. Adjust REB3N bearing and/or cable assembly so that engine idle Ng speed is 56-58% in RUN position.
  5. Set the FCU control rod stop for flight idle Ng speed of 68-70%.
  6. Tighten all jam nuts, verify nuts are cotter pinned and install MS20995C-020 safety wire where required.
