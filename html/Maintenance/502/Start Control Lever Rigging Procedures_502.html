Refer to Pratt & Whitney Maintenance Manual for appropriate engine for general instructions and drawings. Rig Engine Start Control as follows:

  1. Position Start Lever in cockpit in “RUN" position
  2. Adjust REB3N bearing at end of cable so that condition lever on engine is straight up midway between stops.
  3. Pull Start Lever in cockpit all the way back against the quadrant stop. Condition lever on engine should be at the fuel cut-off position.
  4. Perform flight line engine run-up. Adjust REB3N bearing and/or cable assembly so that in “RUN" position engine idle Ng speed is XX-XX%.
  5. Set the FCU control rod stop for FLIGHT IDLE Ng speed of XX-XX%.
  6. Tighten all jam nuts, check nuts for cotters, install safety wire where required.
