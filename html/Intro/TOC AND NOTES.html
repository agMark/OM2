PUBLICATION NOTES

The June 21, 2018 revision to this manual provides procedures and inspection frequencies that were previously in separate manuals (US, Argentina, Australia, and Canada). The result of these changes is a single Owner's Manual for the AT-802/802A series. The inspection frequencies have been harmonized to be consistent for all AT-802/802A operators.

The September 1, 2025 revision of this manual was a significant reorganization. Pages are no longer controlled. Revisions are to be noted by Section which were numbered.
