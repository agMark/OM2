PUBLICATION NOTES

The March 14, 2018 revision to this manual provides procedures and inspection frequencies that were previously in separate manuals (US, Argentina, Australia, Brazil, Canada, Indonesia, and Uruguay). The result of these changes is a single Owner's Manual for the AT-502A/502B/504 series. The inspection frequencies have been harmonized to be consistent for all AT-502A/502B/504 operators.

The March 1, 2026 revision of this manual was a significant reorganization. Pages are no longer controlled. Revisions are to be noted by Section which were numbered.
