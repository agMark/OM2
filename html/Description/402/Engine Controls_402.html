A quadrant on the left-hand side of the cockpit incorporates the POWER lever, the PROPELLER control lever and the START control lever.

There are two types of quadrants:

  • Standard Air Tractor Quadrant
  • Kawak Quadrant

These quadrants are described in the following sections.

All engine controls utilize push-pull cable assemblies. These cable assemblies have very little friction and require no lubrication. See Figure XXX for the layout of engine controls.

The propeller lever may be placed in the full forward position during the start sequence for more convenient movement for the start lever, providing the temperatures are warm. For cold weather starts the prop lever should be aft as specified in the starting instructions, directing initial oil pressure to the engine bearings instead of the propeller dome.
